# Student Profile Website

This is a simple personal profile website designed for college students.  
It showcases the student’s personal information, skills, and contact details in a clean and beginner-friendly layout.

## Features

- **About Me Section** – A brief introduction about the student  
- **Skills Section** – Highlights key technical skills  
- **Contact Section** – Email and phone information  
- **Responsive Design** – Works well on desktop and mobile browsers  

## How to View

1. Open `index.html` in any browser locally  
2. Or visit the live website: [Live Website](https://tanayahiralkar26-commits.github.io/Student--Profile/)
